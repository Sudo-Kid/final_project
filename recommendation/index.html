FTP Recommendation

Jquery/Ajax

There are a lot of jquery plugins available on the net to create an upload page and resolve connecting on the FTP for file updating files on the server.

Example of the plugins are

Ajax file upload - http://www.phpletter.com/Our-Projects/AjaxFileUpload/

and libraries at http://plugins.jquery.com/tag/file/

Although, it is imperative to have a login page for admin for security reasons.

    Another way to resolve this is to use CMS
  • Drupal - is most recommendable given that it can extend the site features into wide and various functions
  • WordPress - as the most user-friendly and ease of use. It is most commonly used by a lot of web designers and website.
